# Лабораторная работа 5: Agile и Scrum

## Проект: Todo List Application

Простое веб-приложение для управления задачами, разработанное с использованием HTML, CSS и JavaScript. Приложение позволяет пользователям добавлять новые задачи, отмечать их как выполненные, фильтровать по статусу и сохранять состояние при перезагрузке страницы. Интерфейс интуитивно понятен и адаптирован для разных устройств.

## Структура проекта:
- `index.html` - главная страница
- `style.css` - стили приложения
- `app.js` - основная логика
- `README.md` - документация
